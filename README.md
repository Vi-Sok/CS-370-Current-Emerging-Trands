# CS-370-Current-Emerging-Trands
CS 370 Module Eight Journal

For Project Two, I built a pirate intelligent agent that learns to solve a maze using Deep Q-Learning. The starter code provided the basic maze/game setup and structure. My main work was implementing the reinforcement learning training behavior, adjusting key settings (like training epochs and update frequency), and verifying the agent could consistently reach the treasure rather than succeeding by chance.

Computer scientists solve real problems by modeling them and creating systems that make good decisions efficiently and reliably. In this course, I approached the project by understanding the rules and constraints, choosing a method (reinforcement learning), implementing the solution, and testing/iterating based on results.

Ethically, I’m responsible for building systems that are reliable and properly tested, being honest about limitations, and considering how automated decision-making can impact end users and organizations.
